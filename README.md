# ⚽ Пинай мяч — онлайн

Многопользовательская браузерная игра про пинание мяча. Каждый игрок управляет
своим кружком, бегает по полю и толкает общий мяч в чужие ворота — как в Haxball.

Сервер на Node.js держит авторитетную физику (мяч и игроки считаются на сервере),
браузеры присылают только ввод и рисуют картинку. Связь — по WebSocket.

## Управление

- Движение — `W` `A` `S` `D` или стрелки
- Удар — `Пробел`
- Красные играют в левые ворота, синие — в правые

## Запуск локально

Нужен [Node.js](https://nodejs.org/) 18+.

```bash
npm install
npm start
```

Открой <http://localhost:3000> в браузере. Чтобы сыграть вдвоём на одном
компьютере — открой адрес в двух вкладках.

## Игра вдвоём по сети

- В одной Wi-Fi сети друзья заходят на `http://<твой-IP>:3000`.
- Через интернет — нужен хостинг с поддержкой WebSocket (например Render)
  или временный туннель (`cloudflared`).

## Файлы

| Файл | Что это |
|------|---------|
| `server.js` | Сервер: физика, комната, счёт, раздача статики |
| `public/index.html` | Клиент онлайн-игры |
| `public/solo.html` | Одиночная версия (без сервера, прицел мышкой) |

## Переменные окружения

- `PORT` — порт сервера (по умолчанию `3000`).
